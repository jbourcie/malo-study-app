# MaloCraft — Map Overlay : Concept & UX

## Rôle
La Map Overlay est l’interface principale côté enfant.
Elle remplace la navigation par pages multiples.

---

## Hiérarchie visuelle

1. Monde (année scolaire)
2. Biomes (matières)
3. Zones (sous-thèmes)

---

## Narration visuelle

| État pédagogique | Représentation |
|------------------|----------------|
| Jamais travaillé | fissuré, terne |
| En cours | échafaudages |
| Reconstruit | propre, lumineux |
| Abandonné | patiné, envahi |

---

## PNJ Guide
Le PNJ est présent dans chaque biome.
Il sert à :
- guider pédagogiquement,
- orienter l’UX,
- proposer une action prioritaire.

Le PNJ déclenche :
- un highlight visuel,
- l’ouverture automatique du bon drawer.

---

## Navigation
- Une seule page Monde
- Drawers latéraux pour le détail
- Sessions lancées via /theme/*

---

## Accessibilité
- zones cliquables larges
- hover visible
- navigation clavier et tactile

---

## Conclusion
La map est :
- un tableau de bord pédagogique,
- une carte de progression,
- un espace narratif vivant.
